== Description ==

Custom Call to Action block which permits the user to choose a background image, a button background color, button arrow color and text attributes



== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/divine-cta` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress




== Screenshots ==

1. Enter a title. You can change the font size and color.
2. Change the width of the title
3. Align the text
4. Change the font size, the font color, the letter spacing and the capitalization
5. Select the border color, type and size
6. Select a URL
7. Select a background image
8. Select the button color settings. You can select the arrow color and the background
9. Select the overlay color and transparency
